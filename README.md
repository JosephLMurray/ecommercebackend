# E-Commerce Backend

## Description

An e-commerce backend that allows for inventory maintenance of Products, Tag, and Categories.

## Table of Contents

- [Installation](#installation)
- [Usage](#usage)
- [Credits](#credits)
- [License](#license)

## Installation

npm install

## Usage

It is an api handled through /api/(products, tags, or categories) routing.
